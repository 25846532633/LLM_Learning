# 常见疑问汇总

这份文件不是在讲“答案”，而是专门整理给我自己复习、自测、自省用的 **问题清单**。  
使用方法建议：
- 第一轮复习：先只看问题，尝试口头回答
- 第二轮复习：结合 notebook 和代码回答
- 第三轮复习：脱离代码，自己重新讲一遍或手写最小实现

# 八、给我自己的复习建议

每轮复习都不要只停留在“看懂”。至少做下面三件事：

1. **口头讲一遍**：不看代码，讲原理和数据流
2. **手写最小实现**：哪怕不完整，也要尝试默写核心 forward
3. **对比前后版本**：回答“它比前一个版本多解决了什么问题”

如果一个问题我答不上来，不要急着看答案，先回 notebook 找：
- 输入输出 shape
- 核心公式
- forward 主流程
- 它相对前一个模块的变化点

这样这份《常见疑问汇总.md》才真正能变成我的复习提纲，而不只是问题清单。
